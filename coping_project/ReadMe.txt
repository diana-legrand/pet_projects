# Impact of Coping Strategy on Employee Performance

## Project Description

Investigated how different coping strategies (problem-focused, emotion-ocused, avoidance) influence employee performance levels. Conducted hypothesis testing (t-tests, ANOVA) to detect differences between coping strategy groups. Built regression models (linear/multiple regression) to predict performance based on coping mechanisms.	

## Tools:
- *pandas*
- *statsmodels*
- *sklearn*
